# חיזוי וניתוח מוקדי סיכון בדרכים בישראל 🚗📊

## תיאור הפרויקט
פרויקט זה נועד לנתח ולחזות מוקדי סיכון המועדים לתאונות דרכים ברחבי ישראל באמצעות מודלים של למידת מכונה (Machine Learning) וניתוח נתונים (Data Analysis). 
מטרת העל של הפרויקט היא לספק כלים תומכי החלטה לרשויות המקומיות, משטרת תנועה ומשרד התחבורה, על מנת לשפר את הבטיחות בדרכים ולהציע פתרונות תשתיתיים מונעים.

## תכונות מרכזיות (Features)
- 🗺️ **זיהוי מרחבי** של אזורים וצמתים בעלי סבירות גבוהה לתאונות.
- 📈 **ניתוח נתונים היסטוריים** של תאונות דרכים בישראל לפי חומרה, שעות, תנאי מזג אוויר וסוג קטלניות.
- 🤖 **מודל חיזוי (Predictive Model)** להערכת רמת הסיכון העתידית בנקודות שונות.
- 📊 **לוח בקרה אינטראקטיבי (Dashboard)** להצגה חזותית ונוחה של התובנות והנתונים כולל מפות חום.

## מקורות נתונים (Data Sources)
הפרויקט מתבסס על מקורות המידע הבאים:
- נתוני הלשכה המרכזית לסטטיסטיקה (למ"ס) אודות תאונות דרכים.
- נתוני GIS ומיפוי של תשתיות כבישים וצמתים בישראל.
- נתוני מזג אוויר היסטוריים.
- נתוני עומסי תנועה (בהתאם לזמינות הנתונים).

## טכנולוגיות כלי פיתוח (Tech Stack)
* **שפת תכנות:** Python
* **ספריות לניתוח נתונים:** Pandas, NumPy
* **למידת מכונה:** Scikit-learn, XGBoost / Random Forest
* **ויזואליזציה:** Matplotlib, Seaborn, Plotly, Folium (למיפוי)
* **סביבת עבודה:** Jupyter Notebook / IDE

## התקנה והרצה (Installation & Usage)

1. **שכפול המאגר (Clone Repository):**
   ```bash
   git clone https://github.com/your-username/israel-road-risk-analysis.git
   cd israel-road-risk-analysis
   ```

2. **יצירת סביבה וירטואלית והתקנת תלויות:**
   ```bash
   python -m venv venv
   # בסביבת Windows:
   venv\Scripts\activate
   # בסביבת Mac/Linux:
   source venv/bin/activate
   
   pip install -r requirements.txt
   ```

3. **הפעלת הפרויקט:**
   ניתן להריץ את מחברות ה-Jupyter שבתיקיית `notebooks/` כדי לראות את שלבי עיבוד הנתונים והאימון, או להריץ את קובץ ההפעלה הראשי (במידה וקיים לוח בקרה).
   ```bash
   jupyter notebook
   # או
   python main.py
   ```

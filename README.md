# חיזוי מוקדי סיכון בתשתיות תחבורה בישראל 🚗📊

## תיאור הפרויקט
מערכת תומכת-החלטה **למנהלים בתחום תשתיות תחבורה** — מזהה ומדרגת אתרי סיכון (צמתים, מקטעי כביש) ברחבי ישראל לפי הסתברות לתאונה חמורה.

המערכת מאומנת על **11,554 תאונות אמיתיות** (נתוני למ"ס PUF 2021) באמצעות מודל Random Forest, ומייצרת לכל מנהל רשימת עדיפויות של אתרי תשתית הדורשים התערבות — לפי ציון סיכון מחושב, סוג הדרך, ומאפייני התאונות ההיסטוריות.

---

## תכונות מרכזיות (Features)

### 🤖 מנוע חיזוי מוקדי סיכון (ML)
- **מודל Random Forest** — מאומן על נתוני תאונות אמיתיים (CBS PUF 2021)
- **הגדרת אתר תשתית** — שילוב ישוב/כביש + סוג דרך (עירוני בצומת, בין-עירוני וכו')
- **פיצ'רים**: שעה, יום, מזג אוויר, מצב כביש, מהירות מותרת, סוג תאונה
- **ציון סיכון 0–100** לכל אתר, מבוסס הסתברות לתאונה קשה/קטלנית

### 🎯 ממשק מנהל — דירוג אתרי תשתית
- **טבלת עדיפויות** — כל האתרים מדורגים לפי ציון סיכון ML עם 🔴/🟡/🟢
- **גורם מוביל** לכל אתר — מה מסביר את הסיכון (שעת שיא, מזג אוויר וכו')
- **מפת חום** גאוגרפית של ריכוזי סיכון
- **סינון** לפי מחוז, סוג דרך, רמת סיכון
- **ייצוא Excel** לדיווח ותכנון תקציבי

### 📊 ניתוח חקרני (EDA)
- **KPI cards** — סה"כ תאונות, נפגעים, תאונות קטלניות וקשות
- **ניתוח זמני** — לפי שעה, יום בשבוע, חודש
- **ניתוח מיקומי** — מחוזות וכבישים עם הכי הרבה תאונות
- **מפה אינטראקטיבית** — פיזור תאונות לפי חומרה
- **סינון** לפי מיקום, חומרה, מזג אוויר

## מקורות נתונים (Data Sources)

| קובץ | מקור | שורות | תיאור |
|------|------|--------|--------|
| `data/accidents_israel_2021_raw.csv` | [למ"ס — PUF 2021](https://data.gov.il/he/dataset/2021-puf) | 11,554 | נתוני תאונות אמיתיים, גולמיים, עם קודי CBS |
| `israel_road_accidents_simulated.csv` | נתונים סימולטיביים | ~1,000 | שימוש לבדיקות ופיתוח EDA |

### עמודות מרכזיות בנתוני הגלם (CBS PUF 2021)
| עמודה | תיאור |
|--------|--------|
| `HUMRAT_TEUNA` | חומרת התאונה (1=קטלנית, 2=קשה, 3=קלה) |
| `SUG_DEREH` | סוג הדרך (1=עירוני בצומת ... 4=בין-עירוני) |
| `MEZEG_AVIR` | מזג אוויר (1=בהיר, 2=גשם קל, 3=גשם ...) |
| `KVISH1` | מספר הכביש |
| `SEMEL_YISHUV` | קוד ישוב (CBS) |
| `SHNAT_TEUNA / HODESH_TEUNA / SHAA` | שנה / חודש / רבע שעה ביום |
| `X, Y` | קואורדינטות ITM (EPSG:2039) — יש להמיר ל-WGS84 |
| `MAHOZ` | מחוז (1=ירושלים ... 6=דרום) |

## התקנה והרצה (Installation & Usage)

### דרישות מוקדמות
- Python 3.12+

### התקנה מהירה

```bash
git clone https://github.com/amarzidan1/git-clone-https---github.com-Ariel-University-AI-project-amarzidan1.git
cd git-clone-https---github.com-Ariel-University-AI-project-amarzidan1
pip install -r requirements.txt
```

### הרצת האפליקציה

```bash
streamlit run eda_app.py
```

או לחץ פעמיים על `run_app.bat` (Windows בלבד) — מתקין ומריץ אוטומטית.

האפליקציה תיפתח בדפדפן: **http://localhost:8501**

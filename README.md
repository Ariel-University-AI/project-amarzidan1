# חיזוי וניתוח מוקדי סיכון בדרכים בישראל 🚗📊

## תיאור הפרויקט
פרויקט זה נועד לנתח ולחזות מוקדי סיכון המועדים לתאונות דרכים ברחבי ישראל באמצעות מודלים של למידת מכונה (Machine Learning) וניתוח נתונים (Data Analysis).
מטרת העל של הפרויקט היא לספק כלים תומכי החלטה לרשויות המקומיות, משטרת תנועה ומשרד התחבורה, על מנת לשפר את הבטיחות בדרכים ולהציע פתרונות תשתיתיים מונעים.

---

## תכונות מרכזיות (Features)

### 📊 ניתוח חקרני (EDA)
- **KPI cards** — סה"כ תאונות, נפגעים, תאונות קטלניות וקשות, ממוצע רכבים לתאונה
- **התפלגות קטגוריאלית** — חומרת תאונות, מזג אוויר, עומס תנועה
- **ניתוח זמני** — לפי שעה ביום, יום בשבוע, חודש בשנה
- **ניתוח מיקומי** — ערים עם הכי הרבה תאונות ונפגעים
- **מפה אינטראקטיבית** — פיזור תאונות על מפת ישראל לפי חומרה
- **סיכום סטטיסטי** — מספרי וקטגורי לכל עמודה

### 🛡️ המלצת כביש בטוח
- בחירת **מוצא ויעד** מרשימת כל הערים והכבישים
- **ציון סיכון** לכל מיקום (0–100) לפי חומרת תאונות היסטוריות
- **המלצה ברורה**: ✅ בטוח / ⚠️ בינוני / 🚨 מסוכן
- **טבלת דירוג** צבעונית של כל הכבישים
- **מפת סיכון** — ירוק=בטוח, אדום=מסוכן

### 🔧 סינון אינטראקטיבי
- סינון לפי עיר/כביש, חומרת תאונה, מזג אוויר

## מקורות נתונים (Data Sources)

| קובץ | מקור | שורות | תיאור |
|------|------|--------|--------|
| `data/accidents_israel_2021_raw.csv` | [למ"ס — PUF 2021](https://data.gov.il/he/dataset/2021-puf) | 11,554 | נתוני תאונות אמיתיים, גולמיים, עם קודי CBS |
| `israel_road_accidents_simulated.csv` | נתונים סימולטיביים | ~1,000 | שימוש לבדיקות ופיתוח EDA |

### עמודות מרכזיות בנתוני הגלם (CBS PUF 2021)
| עמודה | תיאור |
|--------|--------|
| `HUMRAT_TEUNA` | חומרת התאונה (1=קטלנית, 2=קשה, 3=קלה) |
| `SUG_DEREH` | סוג הדרך (1=עירוני בצומת ... 4=בין-עירוני) |
| `MEZEG_AVIR` | מזג אוויר (1=בהיר, 2=גשם קל, 3=גשם ...) |
| `KVISH1` | מספר הכביש |
| `SEMEL_YISHUV` | קוד ישוב (CBS) |
| `SHNAT_TEUNA / HODESH_TEUNA / SHAA` | שנה / חודש / רבע שעה ביום |
| `X, Y` | קואורדינטות ITM (EPSG:2039) — יש להמיר ל-WGS84 |
| `MAHOZ` | מחוז (1=ירושלים ... 6=דרום) |

## התקנה והרצה (Installation & Usage)

### דרישות מוקדמות
- Python 3.12+

### התקנה מהירה

```bash
git clone https://github.com/amarzidan1/git-clone-https---github.com-Ariel-University-AI-project-amarzidan1.git
cd git-clone-https---github.com-Ariel-University-AI-project-amarzidan1
pip install -r requirements.txt
```

### הרצת האפליקציה

```bash
streamlit run eda_app.py
```

או לחץ פעמיים על `run_app.bat` (Windows בלבד) — מתקין ומריץ אוטומטית.

האפליקציה תיפתח בדפדפן: **http://localhost:8501**
